Path Planning by implementing APF
This code implements APF(artificial potential fields), which a method used to formulate obstacle and goal interactions in a robot’s path. In this implementation, obstacles act as repulsive forces where targets act as attractive ones. The code is written for ROS environment, and experimented on real robots.
